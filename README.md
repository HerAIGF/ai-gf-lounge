## Quickstart (local)
1. Copy `.env.example` to `.env` and fill keys.
2. Install deps:
   ```
   cd herai
   npm install
   ```
3. Run locally:
   ```
   npm run dev
   ```
4. Follow docs/DEPLOY.md to deploy to Supabase & Vercel.
